Detailed Description

template<typename Matrix, typename T, size_t Dims = 2>
class otus_sparse_matrix::Indexer< Matrix, T, Dims >

Класс помощник для SparseMatrix, обеспечивающий функциональность оператора [] для многомерных случаев

Template Parameters
Matrixтип данных с разреженной матрицей
Tтип элемента, хранящийся в матрице
Dimsколичество измерений в матрице. По умолчанию 2 измерения.
