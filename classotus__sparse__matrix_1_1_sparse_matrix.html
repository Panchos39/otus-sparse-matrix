Detailed Description

template<typename T, T EmptyValue, int Dims = 2>
class otus_sparse_matrix::SparseMatrix< T, EmptyValue, Dims >

Класс proxy для std::map, обеспечивающий функциональность разреженной матрицы

Template Parameters
Tтип данных, хранящийся в разряженной матрице
EmptyValueзначение типа T, которое будет считаться не заполненным для разреженной матрицы
Dimsколичество измерений в матрице. По умолчанию 2 измерения.
